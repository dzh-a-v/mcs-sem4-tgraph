# Биномиальное распределение
## Справочник Вадзинского Р.П. — Раздел 2.4

---

## 1. Основные характеристики

### 1.1 Ряд распределения

$$p(x) = C_n^x p^x q^{n-x}, \quad x = 0, 1, \ldots, n$$

**Условия:**
- $n \geq 1$ — целое
- $0 < p < 1$
- $q = 1 - p$

---

### 1.2 Функция распределения

$$
F(x) = 
\begin{cases} 
0, & x \leq 0 \\
\sum\limits_{i=0}^{k} C_n^i q^{n-i}, & k < x \leq k+1, \quad k = 0, 1, \ldots, n-1 \\
1, & x > n
\end{cases}
$$

---

### 1.3 Производящая функция

$$\varphi_x(t) = (q + pt)^n$$

---

### 1.4 Характеристическая функция

$$\chi_x(t) = (q + pe^{it})^n$$

---

### 1.5 Математическое ожидание

$$\bar{x} = np$$

---

### 1.6 Мода

$$
\hat{x} = 
\begin{cases} 
[(n+1)p], & (n+1)p \text{ — не целое} \\
(n+1)p - 1, \\
(n+1)p, & (n+1)p \text{ — целое}
\end{cases}
$$

---

### 1.7 Дисперсия

$$D_x = npq$$

---

### 1.8 Стандартное отклонение

$$\sigma_x = \sqrt{npq}$$

---

### 1.9 Коэффициент вариации

$$v_x = \sqrt{\frac{q}{np}}$$

---

### 1.10 Асимметрия

$$Sk = \frac{q - p}{\sqrt{npq}}$$

---

### 1.11 Эксцесс

$$Ex = \frac{1 - 6pq}{npq}$$

---

## 2. Моменты

### 2.1 Начальные моменты

$$m_2 = np[(n-1)p + 1]$$

$$m_3 = np[(n-1)(n-2)p^2 + 3(n-1)p + 1]$$

$$m_4 = np[(n-1)(n-2)(n-3)p^3 + 6(n-1)(n-2)p^2 + 7(n-1)p + 1]$$

$$m_{r+1} = np\left[m_r + \frac{q}{n}\frac{dm_r}{dp}\right]$$

---

### 2.2 Центральные моменты

$$\mu_1 = 0$$

$$\mu_2 = npq$$

$$\mu_3 = npq(q - p)$$

$$\mu_4 = npq[1 + 3pq(n - 2)]$$

---

## 3. Типичная интерпретация

1. $X$ — число успехов в серии из $n$ независимых испытаний Бернулли с вероятностью успеха $p$ и вероятностью неудачи $q = 1 - p$ в каждом испытании

2. $X$ — число меченых элементов в случайной выборке с возвращением объёма $n$, извлечённой из генеральной совокупности объёма $N$, содержащей $M = pN$ меченых элементов

---

## 4. Соотношения между распределениями

### 4.1 Симметрия биномиальных распределений

Для биномиальных распределений с параметрами $n, p$ и $n, 1-p$ справедливы соотношения:

$$(1) \quad p(x; n, p) = p(n - x; n, 1 - p)$$

$$(2) \quad P\{X(n; p) \leq x\} = 1 - P\{X(n; 1 - p) \leq n - x - 1\}$$

---

### 4.2 Сумма независимых биномиальных величин

Сумма $s$ независимых биномиальных случайных величин с параметрами $n_i, p$ $(i = 1, 2, \ldots, s)$ подчиняется биномиальному закону распределения с параметрами $v, p$, где:

$$v = n_1 + n_2 + \ldots + n_s$$

---

### 4.3 Связь с F-распределением Фишера-Снедекора

Для биномиальной случайной величины $X(n, p)$ и случайной величины $Y(2(x+1), 2(n-x))$, имеющей F-распределение Фишера-Снедекора с числом степеней свободы $v_1 = 2(x+1)$, $v_2 = 2(n-x)$, справедливо равенство:

$$P\{X(n, p) \leq x\} = 1 - P\left\{Y(2(x+1), 2(n-x)) \leq \frac{p(n-x)}{(1+x)(1-p)}\right\}$$

$$= P\left\{Y(2(n-x), 2(x+1)) < \frac{(1+x)(1-p)}{p(n-x)}\right\}$$

---

### 4.4 Предельный переход к распределению Пуассона

При $n \to \infty$, $p \to 0$, так что $np = \mu = \text{const}$, биномиальное распределение стремится к распределению Пуассона с параметром $\mu$.

---

### 4.5 Связь с геометрическим и отрицательным биномиальным распределениями

Рассмотрена в п. 2.3 справочника.

---

### 4.6 Выбор между моделями

| Модель | Свойство дисперсии |
|--------|-------------------|
| Биномиальная | дисперсия меньше среднего |
| Пуассоновская | дисперсия равна среднему |
| Отрицательная биномиальная | дисперсия больше среднего |

## 5. Оценивание параметров

### 5.1 Метод максимального правдоподобия

Оценки максимального правдоподобия параметров $n$ и $p$ определяются путём решения системы уравнений:

$$(3) \quad \begin{cases}
n^* p^* = \bar{x}^* \\
\sum\limits_{k=0}^{x_{max}-1} \frac{v_k}{n^* - k} = -m \ln\left(1 - \frac{\bar{x}^*}{n^*}\right)
\end{cases}$$

**Где:**
- $v_k$ — число элементов выборки $x_1, x_2, \ldots, x_m$, превышающих $k$
- $x_{max} = \max(x_1, x_2, \ldots, x_m)$ — максимальный элемент выборки
- $\bar{x}^*$ — выборочное среднее

**Правила округления:**
- Если $n^*$, полученное при решении системы (3), больше $x_{max}$, то в качестве оценки параметра $n$ рекомендуется взять целое число, ближайшее к $n^*$
- Если же $n^* < x_{max}$, то в качестве оценки $n$ следует взять $x_{max}$

**Если параметр $n$ известен**, ОМП параметра $p$ определяется по формуле:

$$p^* = \frac{\bar{x}^*}{n}$$

---

### 5.2 Метод моментов

$$n^* = \frac{(\bar{x}^*)^2}{\bar{x}^* - S_x^2}$$

$$p^* = 1 - \frac{S_x^2}{\bar{x}^*}$$

**Важно:** При $\bar{x}^* < S_x^2$ оценка $n^*$ становится отрицательной. Это указывает на то, что в данном конкретном случае биномиальная модель неприменима.

## 6. Генерирование случайных чисел

### 6.1 Алгоритм 1

Блок-схема алгоритма генерирования биномиальных случайных чисел.

```
┌─────────────────┐
│   Ввод: n, p    │
└────────┬────────┘
         ▼
┌─────────────────┐
│ q := 1 - p      │
│ c := p / q      │
│ p := q^n        │
└────────┬────────┘
         ▼
┌─────────────────┐
│ r := rand()     │
└────────┬────────┘
         ▼
┌─────────────────┐
│ x := 0          │
└────────┬────────┘
         ▼
┌─────────────────┐
│ r := r - p      │
└────────┬────────┘
         ▼
    ┌────────┐
    │ r < 0? │
    └───┬────┘
        │
   ┌────┴────┐
   │         │
  ДА        НЕТ
   │         │
   ▼         ▼
┌─────┐   ┌─────────────────┐
│ X:=x│   │ x := x + 1      │
└──┬──┘   │ p := p*c*(n+1-x)/x │
   │      └────────┬────────┘
   │               │
   │               │
   ▼               │
                   │
                   │
                   ▼
                (возврат к r := r - p)
```

**Описание:** Алгоритм 1 реализует стандартный способ имитационного моделирования дискретных случайных величин.

## 8. Техника вычислений

### 8.1 Рекуррентное соотношение для вероятностей

Последовательные значения вероятностей $p(x)$ связаны между собой соотношением:

$$p(x) = p(x-1) \cdot \frac{p}{q} \cdot \frac{n + 1 - x}{x}, \quad x = 1, 2, \ldots, n$$

**где** $p(0) = q^n$

---

### 8.2 Вычисление через неполную бета-функцию

При вычислениях, связанных с биномиальным распределением, могут быть использованы соотношения:

$$P(X \leq x) = \sum_{i=0}^{x} C_n^i p^i (1-p)^{n-i} = I_{1-p}(n-x, x+1) = 1 - I_p(x+1, n-x)$$

$$P(X \geq x) = \sum_{i=x}^{n} C_n^i p^i (1-p)^{n-i} = I_p(x, n-x+1) = 1 - I_{1-p}(n-x+1, x)$$

$$p(x) = P(X = x) = I_p(x, n-x+1) - I_p(x+1, n-x) = I_{1-p}(n-x, x+1) - I_{1-p}(n-x+1, x)$$

**Где:**
$$I_x(u, v) = \frac{1}{B(u, v)} \int_0^x t^{u-1}(1-t)^{v-1} dt$$

— отношение неполной бета-функции.

## 9. Аппроксимации

### 9.1 Нормальная аппроксимация

**Условие:** При $npq > 9$ биномиальное распределение можно аппроксимировать нормальным распределением с математическим ожиданием $np$ и дисперсией $npq$:

$$p(x) \approx \frac{1}{\sqrt{npq}} \varphi\left(\frac{x - np}{\sqrt{npq}}\right) \approx \Phi_0\left(\frac{x + 0.5 - np}{\sqrt{npq}}\right) - \Phi_0\left(\frac{x - 0.5 - np}{\sqrt{npq}}\right)$$

$$P(m_1 \leq X \leq m_2) \approx \Phi_0\left(\frac{m_2 + 0.5 - np}{\sqrt{npq}}\right) - \Phi_0\left(\frac{m_1 - 0.5 - np}{\sqrt{npq}}\right)$$

$$P(X \leq m) \approx \frac{1}{2} + \Phi_0\left(\frac{m + 0.5 - np}{\sqrt{npq}}\right)$$

**Максимальная абсолютная ошибка** последнего приближённого равенства меньше $0.140 / \sqrt{npq}$.

**Более мягкие условия** (по источнику [20]): нормальная аппроксимация допускается при $npq > 5$ и $0.1 \leq p \leq 0.9$.

---

### 9.2 Пуассоновская аппроксимация

**Условие:** При $p < 0.1$ можно пользоваться пуассоновским приближением:

$$p(x) \approx \frac{(np)^x}{x!} e^{-np} = \psi(x; np)$$

$$P(m_1 \leq X \leq m_2) \approx \sum_{i=m_1}^{m_2} \frac{(np)^i}{i!} e^{-np} - \sum_{i=m_2+1}^{\infty} \frac{(np)^i}{i!} e^{-np} = \tilde{\Psi}(m_1; np) - \tilde{\Psi}(m_2 + 1; np)$$

---

### 9.3 Функции в системе DERIVE

В системе символьной математики DERIVE имеются функции:

$$\text{BINOMIAL\_DENSITY}(k, n, p) = C_n^k p^k (1-p)^{n-k}$$

$$\text{BINOMIAL\_DISTRIBUTION}(k, n, p) = \sum_{i=0}^{k} C_n^i p^i (1-p)^{n-i}, \quad k = 1, 2, \ldots, n$$

**(см. [41], утилиты PROBABIL.MTH)**
